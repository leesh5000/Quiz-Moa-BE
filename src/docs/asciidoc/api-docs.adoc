= Quiz-App APIs =

== *공통* ==

=== *HTTP 상태 코드* ===

|===
| 상태 코드 | 설명

| `200 OK`
| 성공

| `201 Created`
| 리소스 생성

| `204 No Content`
| 성공 (응답 바디 없음)
정
| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 인증되지 않은 클라이언트

| `403 Forbidden`
| 권한 없음

| `404 Not Found`
| 존재하지 않는 리소스 요청

| `409 Conflict`
| 이미 존재하는 리소스

| `500 Internal Server Error`
| 서버 에러
|===

=== *에러 코드* ===

API 요청이 거부될 경우, 에러 코드와 메세지를 참고하여 원인을 파악할 수 있습니다.

|===
| 에러 코드 | HTTP Status | 메세지

| `A-001`
| `401 Unauthorized`
| 만료된 토큰으로 요청

| `A-002`
| `401 Unauthorized`
| 유효하지 않은 토큰으로 요청

| `A-003`
| `401 Unauthorized`
| 올바르지 않은 Authorization 헤더로 요청

| `A-004`
| `401 Unauthorized`
| Authorization 헤더가 존재하지 않는 경우

| `A-005`
| `401 Unauthorized`
| Authorization 헤더가 Bearer 타입이 아닌 경우

| `A-006`
| `401 Unauthorized`
| 접근 토큰이 존재하지 않은 경우

| `A-007`
| `401 Unauthorized`
| 해당 토큰이 접근 토큰이 아닌 경우

| `A-008`
| `401 Unauthorized`
| 유효하지 않은 갱신 토큰으로 요청

| `A-009`
| `401 Unauthorized`
| 만료된 갱신 토큰으로 요청

| `A-010`
| `401 Unauthorized`
| 이미 로그아웃 처리된 유저

| `A-011`
| `401 Unauthorized`
| 해당 토큰이 갱신 토큰이 아닌 경우

| `O-001`
| `400 Bad Request`
| 지원하지 않는 소셜 플랫폼인 경우

| `O-002`
| `409 Conflict`
| 이미 카카오로 가입된 유저로 다른 소셜 플랫폼을 통해 재가입하는 경우

| `O-003`
| `409 Conflict`
| 이미 구글로 가입된 유저로 다른 소셜 플랫폼을 통해 재가입하는 경우

| `O-004`
| `409 Conflict`
| 이미 네이버로 가입된 유저로 다른 소셜 플랫폼을 통해 재가입하는 경우

| `M-001`
| `409 Conflict`
| 이미 존재하는 이메일로 회원가입 요청

| `M-002`
| `404 Not Found`
| 존재하지 않는 유저

| `M-003`
| `403 Forbidden`
| 해당 유저의 요청에 권한이 없는 경우

| `Q-001`
| `404 Not Found`
| 존재하지 않는 퀴즈를 요청하는 경우

| `Q-002`
| `403 Forbidden`
| 해당 퀴즈의 작성자가 아닌 유저가 퀴즈를 수정하거나 삭제하는 경우

| `A-001`
| `404 Not Found`
| 존재하지 않는 답변을 요청하는 경우

| `V-001`
| `409 Conflict`
| 이미 투표를 한 유저가 다시 투표를 하느 경우
|===
