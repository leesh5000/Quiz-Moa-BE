= Quiz-App APIs

== 공통

=== HTTP 상태 코드

|===
| 상태 코드 | 설명

| `200 OK`
| 성공

| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 인증 안 됨

| `403 Forbidden`
| 권한 없음

| `404 Not Found`
| 존재하지 않는 요청 리소스

| `500 Internal Server Error`
| 서버 에러
|===
