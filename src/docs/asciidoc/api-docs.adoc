== 공통 ==

=== HTTP 상태 코드 ===

|===
| 상태 코드 | 설명

| `200 OK`
| 성공

| `201 Created`
| 리소스 생성

| `204 No Content`
| 성공 (응답 바디 없음)
정
| `400 Bad Request`
| 잘못된 요청

| `401 Unauthorized`
| 인증되지 않은 클라이언트

| `403 Forbidden`
| 권한 없음

| `404 Not Found`
| 존재하지 않는 리소스 요청

| `409 Conflict`
| 이미 존재하는 리소스

| `500 Internal Server Error`
| 서버 에러
|===
