O-003

409 Conflict

이미 구글로 가입된 유저로 다른 소셜 플랫폼을 통해 재가입하는 경우

O-004

409 Conflict

이미 네이버로 가입된 유저로 다른 소셜 플랫폼을 통해 재가입하는 경우

Q-002

403 Forbidden

A-001

404 Not Found

존재하지 않는 답변을 요청하는 경우

V-001

409 Conflict
